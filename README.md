
# Coffee delivery application  - ReactJS   

ReactJS - Implementing a Coffee delivery application for desktop.

- ReactJS
- Typescript
- Phosphor-react 
- styled components
- react router-dom
- context
- provider
- immer
- reduce
- react-hook-form
